My recommendation for your course project

I would teach students this exact flow:

✅ Access Token → Redux (memory only, not persisted)
✅ Refresh Token → HttpOnly Secure Cookie
✅ Axios interceptor to automatically refresh expired access tokens
✅ Redux repopulated after a successful /refresh
✅ Automatic logout if refresh fails
✅ Short-lived access tokens (10–15 minutes)
✅ Longer-lived refresh tokens (e.g., 7 days), with rotation if you want to demonstrate a more advanced setup
